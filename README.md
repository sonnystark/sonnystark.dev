## Getting Started

### Prerequisites

- Node.js (or Bun)
- npm, yarn, or bun

### Installation

```sh
npm install
# or
yarn install
# or
bun install
```

### Development

```sh
npm run dev
# or
yarn dev
# or
bun dev
```

### Build

```sh
npm run build
# or
yarn build
# or
bun build
```

### Preview

```sh
npm run preview
# or
yarn preview
# or
bun preview
```
